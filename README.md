### Instalação

1. #### _Clone o projeto via https_
```
git clone https://github.com/romariocavalcante/Portfolio-React.git
```
2. #### _Instale as dependências via npm_
```
npm install
```
2. #### _Visualize o projeto
```
npm run dev
```

yarn dev
